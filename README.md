Project: **Ovarian Cancer Subtype Classification**

The UBC Ovarian Cancer subtypE clAssification and outlier detectioN (UBC-OCEAN) competition aims to classify ovarian cancer subtypes using machine learning models trained on a vast dataset of histopathology images sourced from over 20 medical centers worldwide.
